# Основы префабов

**Префаб (Prefab)** — это шаблон GameObject, сохранённый в виде ассета в папке Project. Любое количество **инстансов** (экземпляров) префаба в сцене наследует свойства ассета и автоматически обновляется при его изменении.

## Создание и редактирование префаба

**Создание:**
1. Настроить GameObject в сцене (добавить компоненты, дочерние объекты)
2. Перетащить его из иерархии в папку Project
3. В сцене останется инстанс префаба (иконка синего куба в иерархии)

**Редактирование в Prefab Mode:**
1. Двойной клик по ассету в Project — открывается изолированный Prefab Mode
2. Или нажать стрелку `>` рядом с инстансом в иерархии
3. Изменения сразу отражаются на всех инстансах в сценах
4. Нажать **← Back** в верхнем левом углу для выхода

**Отличие Prefab Asset от Prefab Instance:**

| | Prefab Asset | Prefab Instance |
|-|-------------|-----------------|
| Где находится | Папка Project | Сцена / другой префаб |
| Иконка | Синий куб | Синий куб в иерархии |
| Изменения | Применяются ко всем инстансам | Только к данному инстансу (Override) |

## Применение и откат изменений

При изменении инстанса в сцене свойства подсвечиваются жирным — это **Override**.

**Через инспектор:**
- Нажать правой кнопкой на изменённое поле → **Apply to Prefab** / **Revert**

**Через кнопку Overrides в верхней части инспектора:**
- **Apply All** — применить все изменения инстанса к ассету
- **Revert All** — откатить все изменения инстанса до значений ассета

```csharp
// Применение и откат через код (только в Editor-скриптах)
#if UNITY_EDITOR
using UnityEditor;

// Применить изменения инстанса к ассету
PrefabUtility.ApplyPrefabInstance(instance, InteractionMode.UserAction);

// Откатить инстанс до состояния ассета
PrefabUtility.RevertPrefabInstance(instance, InteractionMode.UserAction);
#endif
```

## Распаковка префаба

После распаковки объект теряет связь с ассетом и становится обычным GameObject.

**Через UI:** ПКМ на инстансе в иерархии → **Prefab → Unpack** (один уровень) или **Unpack Completely** (все вложенные префабы).

```csharp
#if UNITY_EDITOR
// Распаковать один уровень
PrefabUtility.UnpackPrefabInstance(
    instance,
    PrefabUnpackMode.OutermostRoot,
    InteractionMode.UserAction
);

// Распаковать полностью (включая вложенные префабы)
PrefabUtility.UnpackPrefabInstance(
    instance,
    PrefabUnpackMode.Completely,
    InteractionMode.UserAction
);
#endif
```

> ⚠️ Распакованный объект больше не получает изменения от ассета. Используйте распаковку только когда нужна полная независимость объекта.
